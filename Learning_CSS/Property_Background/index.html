Propriedade Background

A propriedade backgound Serve tanto para colocar cor a um elemento quanto para adicionar uma imagem

A propriedade backgoundtem varios parametros como:

  • color:" "

    • Define uma cor para o fundo da tela

  • image: url: ("")

    • adiciona um enderedo de uma imagem que esta salvo em uma pasta

    para retornar uma pasta pra tras basta coloca (.) a quantidade de ponto é a quantidade de pastas que voce retorna (ex: url:(".../estou_voltando_3_pastas_pra_tras"))

  • size:" "

    • Define o tamanho da imagem ou da cor onde o fundo vai ser adicionado

  • repeat:" "

    • Define se a imagem vai ser repetida ou nao, e se sim é possivel escolher os lados

    1. No-repeat - para quando a imagem chegar ao fim ela não se repetir
    2. Repeat - Para quando a imagem chegar no seu final ela se repetir
    3. repeat-x/y - Para se repetir em uma direcao, ou X ou Y

  • position:" "

    • Serve para mostrar em que parte da imagem o elemento dará enfase

    1. Center - para focar no centro da imagem em relacao a imagem
    2. Top - Para mostrar mais a parte de cima da imagem
    3. Right - Para mostrar mais a parte da direita da imagem
    4. Left - Para mostrar mais a parte da esquerda da imagem
