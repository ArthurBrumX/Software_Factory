CSS Interno

Basicamente CSS Interno nada mais é doque criar um paragrafo no head com a tag style e comecar a estilizar dentro da pafina do html.


Por exemplo esse paragrafo foi pintado de verde pelo css que esta contido dentro desse mesmo arquivo


Esse outro paragrafo teve sua fonte alterada pelo css que esta contido nesse mesmo arquivo


Esse paragrafo vai ficar menor em relacao aos outros, que tambem foi modificado dentro do mesmo css que esta nesse arquivo mesmo


Em outras palavras voce estiliza dentro do arquivo o css que voce quer pra aquele arquivo e apenas para esse arquivo se ouver qualquer outro arquivo, ele nao herdará as propriedades do css que forem feitas dentro desse arquivo.


O CSS interno tem o seu nivel de prioridade media, pois ele vem depois do inline, entao o seu nivel de prioridade pelo codigo é medio ele só vai ser considerado se o inline nao existir.
