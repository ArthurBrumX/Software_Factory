Property Color


A propriedade Color, faz com que seje possivel realizar a estilizacao da cor do parargrafo ou titulo, nao necessariamente serve apenas para letras, mas é o mais recomendavel o uso e o mais apropriado tambem, com essa propriedade é possivel colorir textos, paragrafos, frases, palavras entre outros...

Existem varios tipos de color:
RGB
# tipo exadecimal
none
