CSS Externo

O CSS Externo diferente do inline ou do interno, ele é em outro arquivo, dedicado apenas para ele para que possa ser feitas todas configuracoes em apenas um lugar, tanto para melhorar o acesso como para aplicar em determinadas paginas html, ja que é só linkar a pagina no html que ja vai estar aplicadas


Esse paragrafo foi deixado todo vermelhor por elementos colocados no arquivo css e linkado a esse arquivo atraves da tag link, ja que essa tank consegui linkar um arquivo no outro, dessa forma foi aplicada essa propriedade color.


O CSS Externo ele tem o nivel de prioridade sendo o mais baixo de acordo com o html, sendo que sua propriedade na maioria das vezes só vai ser levada em consideracao apenas se nao houver nenhum tipo de css ou inline ou interno.
